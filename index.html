台灣高鐵 火車時刻表 台中公車 國道路況

【自行開車】經由國道一號

- 北上方向

下中港交流道(臺中178km)，走臺灣大道往沙鹿清水方向

→ 過沙鹿工業高級中等學校後，於中山路右轉

→ 經勞工育樂中心300公尺後，餐廳在您的右手邊。

- 南下方向

下中清交流道(大雅174km)，走中清路往沙鹿清水方向

→ 過台中航空站後，直行約2公里，斜右轉往清水市區

→ 下中山路匝道左轉，行駛350公尺後，餐廳在您的左前方。

【自行開車】經由國道三號

- 北上 & 南下方向

由沙鹿交流道(沙鹿176km)下走清水沙鹿方向

→ 斜右轉往清水市區

→ 下中山路匝道左轉，行駛50公尺後，餐廳在您的左前方。

【自行開車】經由台61線(西濱快速道路)

- 北上方向

下梧棲出口(154km)，朝梧棲前進

→ 斜右轉接台灣大道八段，直行約1.5公里

→ 左轉上中華路(台1線)匝道，朝清水前進

→ 於思夢樂服飾店右轉長春路

→ 下長春路橋後左轉中山路，行駛800公尺後，餐廳在您的右手邊。

- 南下方向

下清水出口(150km)，朝清水/臺中港/國道4號方向前進

→ 於中清路左轉，直行約1.9公里上高架橋(特一號道路)

→ 下中山路匝道右轉，行駛350公尺後，餐廳在您的左前方。

【於臺中火車站 搭公車】

總路程約1.5小時，需悠遊卡/一卡通(10元)。

步行至臺中車站(臺灣大道)

→ 搭乘303/304/305號線公車(每20分鐘一班，車程80分鐘)

→ 搭至中山星河路口下車後，餐廳在您的右前方。

總路程約2小時，需悠遊卡/一卡通(10元)。

步行至臺中車站牌(臺灣大道)

→ 搭乘9號線公車(每30分鐘一班，車程90分鐘)

→ 搭至清水高中(學園街)下車後，往南步行800m(約10分鐘)，餐廳在您的左手邊。

【於臺中火車站 搭火車】

總路程時間約 1 小時，全票45元，可使用悠遊卡/一卡通。

於臺中火車站搭乘復興號/區間車 (南下成追線)

→ 抵達清水火車站後，中正街步行350m，至中山路右轉後步行300m，左手邊即可到達。

- 火車時刻表資訊 (僅供參考，以台鐵網站公告為準)

臺中-清水 09:44 - 10:35 成追線2606 (南下往通宵)

臺中-清水 11:24 - 12:06 成追線2608 (南下往通宵)

【於臺中火車站 搭計程車】

總路程約1小時。

【於臺中高鐵站 搭公車】

總路程約2小時，需悠遊卡/一卡通。

步行至高鐵臺中站牌

→ 搭乘93號線公車(台中客運公車，每30分鐘一班，車程60分鐘)

→ 搭至中山星河路口下車後，右手邊即可到達。

【於臺中高鐵站 搭火車】

總路程時間約40分鐘，全票33元，可使用悠遊卡/一卡通。

站內轉成至台鐵新烏日車站

→ 搭乘復興號/區間車 (南下成追線)

→ 抵達清水火車站後，中正街步行350m，至中山路右轉後步行300m，左手邊即可到達。

- 火車時刻表資訊 (僅供參考，以台鐵網站公告為準)

新烏日-清水 09:57 - 10:35 成追線2606 (南下往通宵)
